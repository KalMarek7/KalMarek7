# Personal Website - Web Resume

A minimalistic website serving as a digital resume, providing a brief summary of my IT experience, skills, and projects.

## Technologies used

- **[TailwindCSS](https://tailwindcss.com/):** Rapidly build modern websites without ever leaving your HTML.
- **[Highlight.js](https://highlightjs.org/):** The Internet's favorite JavaScript syntax highlighter supporting Node.js and the web.
- **[SVGRepo.com](https://www.svgrepo.com/):** 500.000+ Open-licensed SVG Vector and Icons.
